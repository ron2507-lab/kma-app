Schritt für Schritt
1

Config-Datei & Link erhalten

Du erhältst vom Admin zwei Dinge per WhatsApp oder E-Mail:

🔗 App-Link
https://ron2507-lab.github.io/KMA/
📄 Config-Datei
kma-config.json — speichern in der Dateien-App (iPhone: iCloud oder „Auf meinem iPhone")

Link öffnen → beim ersten Start erscheint automatisch der Einrichtungs-Assistent.

✓ Im Assistenten auf „Aus Datei laden" tippen → kma-config.json auswählen → „Speichern & starten". Die ID wird einmalig gespeichert — danach nie wieder nötig.
2

Speicherort wählen Tab: Setup

Im Tab Setup unter „Speicherort" wählen:

☁ OneDrive
Empfohlen · funktioniert überall
🖥 Lokal
Nur Chrome/Edge · kein iPhone
3

Postfach verbinden

Klick auf „Mit Postfach verbinden" → Microsoft-Login erscheint.

Melde dich mit deinem eigenen Konto an (Outlook, Hotmail, Live, Firmen-Account).

⚠ Microsoft zeigt ggf. „App nicht verifiziert" → Weitere Informationen → Trotzdem fortfahren. Das ist normal bei privaten Apps.
4

OneDrive verbinden nur OneDrive-Modus

Klick auf „Mit OneDrive verbinden" → gleiches Konto wie Postfach oder ein anderes.

✓ Du kannst dasselbe Microsoft-Konto für beides verwenden.
5

Ordner auswählen & Mails laden Tab: Ordner

Tab Ordner öffnen. Die App zeigt alle Ordner deines Postfachs an.

Klick auf die Ordner die du archivieren möchtest (z.B. Posteingang, Gesendete Elemente) — sie werden markiert. Dann „▶ Mails laden" klicken.

✓ Mehrere Ordner können gleichzeitig markiert werden. Nur die markierten Ordner werden archiviert.
6

Archivierung starten Tab: Archivieren

Tab Archivieren öffnen → „▶ Jetzt archivieren" klicken.

Fortschritt läuft in Echtzeit. Jede Mail wird als PDF gespeichert.

🚫 Browser-Tab offen lassen und im Vordergrund lassen — besonders beim lokalen Speichern. Tab-Wechsel oder gesperrter Bildschirm können die Archivierung unterbrechen.
Einstellungen — was muss stimmen
🗑 „Mails nach Archivierung löschen" — aus lassen
Diese Option im Setup nicht aktivieren. Erst wenn alles geprüft und die Archivierung vollständig abgeschlossen ist, kann das Löschen in Absprache mit dem Admin eingestellt werden.
📅 Zeitraum — „Alle Mails" auswählen
Standardmäßig sollte „Alle Mails" aktiv sein — nicht z.B. „nur Mails älter als 30 Tage". Damit wird das gesamte Postfach erfasst.
📦 Batch-Größe — „Alle" belassen
Die Batch-Einstellung bestimmt wie viele Mails auf einmal geladen werden. Standardwert „Alle" lassen — so wird nichts ausgelassen.
⚠ Andere Einstellungen (kleinere Batches, Zeitfilter, automatisches Löschen) sind möglich — bitte vorher mit dem Admin abstimmen. Falsche Einstellungen können dazu führen dass Mails ausgelassen oder vorzeitig gelöscht werden.
Pause & Abbrüche — was passiert
🟡 Automatische Pause
Die App pausiert automatisch wenn der Tab in den Hintergrund wechselt (z.B. du öffnest einen anderen Tab oder sperrst das Gerät). Ein blau-lila Banner erscheint — klick auf „▶ Weiter" um fortzufahren.
🔴 Absturz oder Browser geschlossen
Kein Problem. Beim nächsten Öffnen der App startet automatisch der Resume-Wizard. Er zeigt dir wie viele Mails bereits archiviert wurden und bietet an genau da weiterzumachen — keine Mail wird doppelt gespeichert.
✅ Duplikat-Schutz
Bereits archivierte Mails werden automatisch übersprungen. Du kannst die Archivierung also jederzeit neu starten — sie macht einfach weiter wo sie aufgehört hat.

Häufige Fragen
Sind meine Mails sicher?
Ja. Die App läuft vollständig in deinem Browser — kein Inhalt wird an externe Server gesendet. Die Verbindung zu Microsoft läuft direkt über die offizielle Microsoft Graph API.
Was ist die „Client ID"?
Eine App-ID die Microsoft braucht um das Login-Fenster zu öffnen. Kein Passwort, kein Zugang zu deinen Mails. Dein eigener Microsoft-Login schützt deine Daten.
Muss ich mich jedes Mal neu einloggen?
Ja — Microsoft verlangt das aus Sicherheitsgründen. Deine Einstellungen (Speicherort, Client ID) bleiben aber gespeichert.
Kann ich auf dem iPhone archivieren?
Ja — Link in Safari öffnen, OneDrive-Modus wählen. Lokale Speicherung funktioniert auf iOS nicht. Beim ersten Start die kma-config.json aus der Dateien-App laden — danach bleibt die Einstellung gespeichert.
Die Archivierung hat sich mitten drin aufgehängt — was nun?
Browser schließen und App-Link neu öffnen. Der Resume-Wizard startet automatisch und zeigt dir den letzten Stand. Mit einem Klick geht es weiter — keine Mail geht verloren oder wird doppelt gespeichert.
Ich will alles nochmal archivieren (Journal zurücksetzen)?
Im Tab Archivieren gibt es den Button „Journal zurücksetzen". Danach musst du im Tab Ordner die Mails neu laden, bevor du wieder archivieren kannst.
