Azure App registrieren

Einmalige Einrichtung — dauert ca. 5 Minuten.

1
Azure Portal öffnen
portal.azure.com im Inkognito-Fenster mit dem M365-Konto öffnen
2
Neue App registrieren
App-Registrierungen → Neue Registrierung → Name: KMA → Kontotyp: Alle Organisationen + persönliche Konten
3
Redirect URI setzen
Authentifizierung → Plattform SPA → URI:
https://ron2507-lab.github.io/kma-app/kma.html
4
Berechtigungen hinzufügen
API-Berechtigungen → Microsoft Graph → Delegiert:
Mail.Read · Mail.ReadWrite · Files.ReadWrite · User.Read
5
Client ID kopieren
Übersicht → Anwendungs-ID (Client) → in KMA eintragen
